# MemeNums

A fun project for the Arduino Uno R4 WiFi that displays iconic numbers on the built-in 12x8 LED Matrix.

## Features

- **LED Matrix Display**: Utilizes the built-in 12x8 LED matrix of the Arduino Uno R4 WiFi.
- **Iconic Numbers**:
  - **42**: The "Answer to the Ultimate Question of Life, the Universe, and Everything" from *The Hitchhiker's Guide to the Galaxy*. A staple in computer science culture.
  - **67**: The "Six, Seven" meme, a viral sensation from 2025 that has taken the new generation of programmers by storm.
- **Animation**: Cycles between the two numbers every 2 seconds.

## Hardware Required

- **Arduino Uno R4 WiFi** (Required for the built-in LED Matrix)
- USB Cable for connection to PC

## Software Dependencies

- **PlatformIO** (VS Code Extension)
- **Arduino_LED_Matrix** library (Included in the Arduino Core for Renesas)

## How to Use

1.  Open the project in VS Code with PlatformIO.
2.  Connect your Arduino Uno R4 WiFi.
3.  Upload the code.
4.  Watch the matrix display the legendary numbers!
